Sistem Operasi

Windows merupakan sistem operasi yang termasuk dalam keluarga sistem operasi komputer pribadi yang dikembangkan oleh Microsoft yang menggunakan anatarmuka dengan pengguna berbasis grafik (GUI/Graphical user interface). Sistem Operasi Windows dalam perkembangannya telah memiliki banyak versi diantaranya Windows 1.0, Windows 2.0, Windows 3.0, Windows for Workgroup, Windows NT (Network Technology), Windows 95, Windows 98, Windows 2000, Windows ME (Millennium Edition), Windows XP (Windows Experience), Windows Vista, Windows 7, Windows 8, Windows 8.1, Windows 10.

Microsoft Word

Microsoft Office adalah sebuah paket aplikasi untuk perkantoran yang dikembangkan oleh perusahaan software ternama yaitu Microsoft. Tiap - tiap kebutuhan administrasi perkantoran dapat ditangani oleh aplikasi - aplikasi khusus dalam paket Microsoft Office, diantaranya Microsoft Word, Microsoft Excel, Microsoft PowerPoint, Microsoft Outlook, Microsoft InfoPath, Microsoft FrontPage, Microsoft Visio, Microsoft Access, Microsoft Onenote.

Microsoft Excel

Aplikasi Pengolah Angka adalah program yang difungsikan untuk mengolah data berupa angka yang memiliki berbagai macam fitur serta kegunaannya yang berguna di zaman modern saat ini. Dengan menggunakan aplikasi ini dapat memudahkan kita dalam bekerja dan menyelesaikan tugas. Sehingga dapat sadari atau tidak bahwa penggunaan aplikasi pengolah angka ini menjadi kebutuhan para pekerja kantoran, pelajar, maupun masyarakat lainnya.

Pengelolah Bahan Tayang

Aplikasi pengolah bahan tayang adalah suatu aplikasi yang digunakan untuk membuat file presentasi. Karena fungsinya sebagai alat untuk mengutarakan ide/gagasan, maka program presentasi harus memiliki tampilan yang menarik. Program presentasi dapat memasukkan objek-objek seperti gambar, video dan suara

Desain Grafis

Graphic design is the business of making or choosing marks and arranging them on a surface to convey an idea (Hollis, 1986). Jadi pada dasarnya desain grafis adalah suatu media untuk menyampaikan informasi melalui bahasa komunikasi visual dalam wujud dwimatra ataupun trimatra yang melibatkan kaidah-kaidah estetik. Salah satu media yang digunakan adalah Figma. Figma adalah aplikasi desain yang dapat didesain dengan mudah oleh siapa saja, mulai dari UI/UX website perusahaan hingga mendesain gambar untuk hobi. Figma gratis untuk diunduh, dan meskipun Figma gratis untuk digunakan, Figma memiliki beragam fitur yang tersedia untuk membantu desainer membuat desain yang menarik. Akan tetapi,untuk memnggunakan figma membutuhkan spesifisasi komputer yang cukup tinggi.
